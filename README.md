## Libraries

MONGODB

```bash
npm install --save-dev @types/mongoose
npm install -D @types/mongoose
```

GRAPHQL

```bash
npm install @nestjs/graphql graphql-tools graphql apollo-server-express
```

```bash
npm install --save class-validator
```

```bash
npm install type-graphql
```

## Installation

Install mongodb and create the data base with name cl_movies.

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
